# ReferralFlow Ontario

A professional, accessible web app for Ontario primary care referral checking and inbound specialist-note summaries.

## Files
- `index.html` - app shell
- `styles.css` - visual design
- `app.js` - UI logic and analysis
- `sample-data.js` - synthetic sample documents

## How to run
Open `index.html` in a browser.

## Notes
- Built for synthetic demo data only.
- No live EMR or patient data.
- Designed for hackathon review and team editing.
